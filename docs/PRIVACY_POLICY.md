# MasilPet 개인정보 처리방침

시행일: 2026년 6월 4일

MasilPet은 부산 지역 탐험과 마실펫 성장을 위해 필요한 최소한의 위치·진행 정보를 처리합니다. 이름, 이메일, 전화번호, 소셜 계정은 요구하지 않으며, TourAPI 키는 앱에 포함하지 않습니다.

## 1. 처리하는 정보

- Firebase 익명 인증 식별자
- 체크인 검증에 필요한 현재 위치 좌표, POI와의 거리, 위치 확인 시간, 체크인 시간
- 마실펫, 알, 보상, 도감, 교감 등 앱 진행 데이터
- 기기 내 이어하기를 위한 로컬 진행 저장 데이터

## 2. 이용 목적

- 최근 15분 안에 확인한 위치를 기준으로 150m 체크인 가능 여부 확인
- 주변 POI 목록 표시와 지역 탐험 보상 제공
- 마실펫 성장, 알 부화, 도감 수집률 동기화
- 앱 오류 대응과 서비스 품질 개선

## 3. 보관과 삭제

Firebase에 저장된 익명 사용자 진행 데이터는 서비스 운영 목적에 필요한 기간 동안 보관합니다. 사용자는 앱의 `내 정보` 화면에서 `진행도 초기화`를 실행해 기기 내 로컬 진행 데이터와 온라인 진행 데이터를 초기화할 수 있습니다. Firebase 연결이 준비되지 않은 상태에서는 기기 내 로컬 진행 데이터만 초기화됩니다. 브라우저의 사이트 데이터 삭제 기능을 사용해도 기기 내 로컬 진행 데이터가 삭제됩니다.

## 4. 제3자 서비스

- Firebase Auth, Firestore, Functions, Hosting: 인증, 진행도 저장, 서버 검증, 웹 배포
- TourAPI: 부산 지역 POI 동기화
- OpenStreetMap 타일: 지도 표시와 저작권 고지

## 5. 보안

사용자 진행도 쓰기는 Cloud Functions를 통해 처리하며, Firestore 클라이언트 직접 쓰기는 제한합니다. 공용 지역 데이터 동기화와 TourAPI 호출은 운영자 권한이 있는 계정으로만 수행합니다.

## 6. 문의

공모전 제출·운영 관련 문의는 제출자가 공지한 연락 채널을 통해 받습니다.
